# Auto Swipe

身内利用向けの、手動で開始・停止するAndroid自動スワイプアプリです。

## APKを入れる人向け

1. GitHub Releasesから `autoswipe-debug.apk` をダウンロードします。
2. Androidの確認画面に従ってAPKをインストールします。
3. 初回起動時に配布者から共有されたパスコードを入力します。
4. アプリ内の「ユーザー補助設定を開く」から `Auto Swipe Service` を有効にします。
5. 方向、間隔、スワイプ速度を選び、「開始」を押します。
6. 3秒以内に対象アプリへ切り替えると、自動スワイプが始まります。
7. 画面上の小さな操作パネルでスワイプ/タップを切り替えます。
8. 操作パネルの「位置選択」から、スワイプまたはタップの実行位置を選びます。選択中は矢印またはポインターが表示されます。
9. 「T」ボタンでタイマーを選びます。OFF、1分、5分、10分、30分を切り替えられます。
10. 「ロック」をONにすると、タイマー終了時に画面ロックを実行します。
11. 操作パネルから開始・停止できます。パネルはドラッグで移動できます。

## 開発者向け

1. Android Studioでこのフォルダを開きます。
2. 必要に応じてAndroid SDK 35をインストールします。
3. アプリをビルドして端末へインストールします。

## 注意

- Google Playに公開せずAPKで直接配布する想定です。
- ユーザー補助サービスを使うため、端末ごとに本人が明示的に有効化する必要があります。
- Google Play公開を考える場合、ユーザー補助APIの用途ポリシー確認が必要です。
- パスコードはカジュアルな利用制限です。強いアクセス制御が必要な用途には向きません。
